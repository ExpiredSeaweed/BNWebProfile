Welcome to the

Mondrian Project

A front-end project focused on creating a visually striking digital representation of Piet Mondrian's signature geometric art. This work demonstrates my ability to meticulously replicate complex designs using HTML and CSS while focusing on CSS Flexbox and Grid display manipulation. With each project demands more complicated column, row and grid area placement.

Pieter Cornelis "Piet" Mondriaan

Project 1

My first attempt at replicating a Mondrian composition. This foundational exercise established the basic use of a CSS Grid within the classic 1:1 square aspect ratio, focusing on simple row and column division.

Reference picture for Project 1

Project 2

This project increased the complexity by requiring more detailed CSS Grid management. This iteration maintained the 1:1 square ratio but introduced more numerous and varied row and column tracks to build a denser composition.

Reference picture for Project 2

Project 3

This composition pushed the limits of basic grid structures within the 1:1 square ratio. This piece demanded sophisticated grid area and column-spanning techniques to accurately replicate a highly detailed Mondrian composition.

Reference picture for Project 3

Project 4

My fourth Mondrian project tackles a significant shift in composition: moving from the previous 1:1 square layouts to a taller, non-square aspect ratio. This piece required a more complex grid structure and precise control to replicate the intricate reference painting within the new canvas dimensions.

Reference picture for Project 4

Project 5

My most challenging replication yet. Demanding mixes of few different CSS Grid techniques, including intricate combinations of columns, rows, and grid areas to accurately map the reference painting's irregular and nested block structure.

Reference picture for Project 5

Project Redux

Project Redux is the culmination of my Mondrian series. Instead of replicating a painting, I used this project to create my own original composition, applying my accumulated CSS Grid knowledge in a self-directed challenge. This piece highlights the technical skills learned and the true genius of Mondrian's balance in comparison to my first attempt at the style.
